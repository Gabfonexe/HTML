Fundamentos do HTML

O HTML é a linguagem padrão para a criação de páginas da Web. As tags HTML são como instruções para o navegador, definindo a estrutura e o conteúdo de uma página.

Tags Básicas de Formatação:

  • h1 até h6 : Usadas para definir cabeçalhos em diferentes níveis de importância. h1 é o mais importante e h6 o menos importante.
  • p: Define um parágrafo.
  • mark: Marca um texto com um destaque visual, geralmente com fundo amarelo.
  • small: Diminui o tamanho da fonte do texto.
  • i: Define texto em itálico.
  • u: Sublinha o texto.
  • strong: Define texto em negrito.
  • sub: Define texto como subscrito (abaixo da linha).
  • sup: Define texto como sobrescrito (acima da linha).

Tags Básicas de Listas:

  • ol: Cria uma lista ordenada (numerada).
  • ul: Cria uma lista não ordenada (com marcadores).
  • li: Define um item em uma lista.

Tags Básicas de Links e Divisões:

  • a: Cria um hyperlink, ou seja, um link para outra página.
  • hr: Insere uma linha horizontal.

Tags Básicas de Citação e Abreviação:

  • blockquote: Cria uma citação longa.
  • abbr: Define uma abreviação.

Tags Adicionais:

  • font: Não recomendado para uso moderno. Altera a fonte, cor e tamanho do texto. O CSS é a forma preferida para estilizar fontes.
  • del: Indica texto que foi deletado.
